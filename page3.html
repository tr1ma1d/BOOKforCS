Основы типов

  • Все типы происзодят от System.Object. Он содержит минимальный набор типов(базовые).
  • new выполняет 4-е с-ва: 1) Вычисление байтов. 2) Выделение памяти. 3) Инициализация указателя. 4) Обращение к конструктора.
  • Главная особенность при приведении типов CLR - безопасность.
  • У производных классов. Нужно явное преобразование типов. От главных не требуется преобразование.
  • Также для приведения можно использовать is(сравнивает) и as (Как явное приведение return number)
  • Пространство имен с помощью using. Для компилятора - средство, позволяющие удлинить
  • При одинаковых методах указываем приоритет. Wintellect.Widget(Если нужен Microsoft, то Microsoft).

    • Память

  • В основном выделятся память для реализации типа. Стек(В размере 1Мб).
  • Входной код (prologue code) - Инициализация до начала работы кода.
  • Выходной код (epilogue code) - очистка после реализации.
